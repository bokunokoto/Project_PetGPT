# Project_PetGPT

반려동물 생애주기 관리 웹 서비스. 사용자 위치를 기반으로 주변의 적절한
가게·시설·정보를 연결해주는 플랫폼.

### 주의사항!!!!!!
DB 필요 시 Sqlite3, 또는 .csv 형태만 사용할 것. Json이나 외부 DB(redis, postgresql) 사용하지 마세요

## 페이지별 요구사항

### 1. 입양 (가족 찾기)
- [ ] 사용자 라이프스타일 입력 (주거, 가족, 활동량, 알러지)
- [ ] 조건 기반 품종·성향 추천
- [ ] 위치 기반 보호소·입양센터 안내
- [ ] 동물보호관리시스템 공공데이터 연동
- [ ] 품종별 특성 정보 제공 (수명, 양육비, 주의 질환)
- [ ] 입양 전 체크리스트
- [ ] 입양 후 적응 가이드 (1주, 1개월, 3개월)
- [ ] 예상되는 비용

### 2. 식단 (맞춤 영양)
- [ ] 반려동물 프로필 등록
- [ ] RER·MER 기반 하루 권장 칼로리 계산
- [ ] 건강 고민별 추천 영양 성분 안내
- [ ] 시중 사료 비교 검색
- [ ] 반려동물 위험 음식 목록
- [ ] 다묘·다견 프로필 동시 관리
- [ ] 체중 변화 그래프
- [ ] 간식 칼로리 계산
- [ ] 비만도(BMI 유사 지표) 분석
- [ ] 음수량 계산

### 3. 용품점 찾기
- [ ] 시/도 → 시군구 → 동 단위 위치 입력
- [ ] 지도와 목록으로 매장 표시
- [ ] 영업시간·취급 품목 정보
- [ ] 서비스 필터링 (미용, 호텔 등)
- [ ] 전화 걸기·길찾기 외부 연결
- [ ] 사용자 평점·리뷰
- [ ] 가격 비교 기능
- [ ] 온라인 쇼핑몰 연동
- [ ] 반려동물 유치원 및 행동교정 예약

### 4. 건강 수첩
- [ ] 1. 건강 관련 일정 및 기록 관리 (예방접종, 구충 등)
      - 현재 날짜와 일정 날짜의 차이 계산해서 D-day 표시
      - 스케줄러 (ex.Celery, node-cron)를 사용하여 예정일 전날 알림 방송
      - 약 복용 알림 (사용자가 '복용 완료'를 누르면 History 테이블에 기록하여 복용 준수율 통계 제공, 매일 정해진 시간 푸시 알림)
- [ ] 2. 건강 데이터 분석 및 시각화 (진단, 처방, 비용)
      - 진료 기록 CSV·PDF 내보내기 (시간순으로 정렬된 깔끔한 템플릿 제작)
      - 체중 변화 그래프 (데이터 입력 시 시각화)
- [ ] 3. 위치 및 네트워크 기능
      - 현재 위치 기반 동물병원 찾기


### 5. 마지막 안녕
- [ ] 위치 기반 장례식장 안내
- [ ] 시설 유형·서비스 정보 (개별 화장, 봉안당 등)
- [ ] 정식 허가 여부 표시
- [ ] 비대면 상담 신청
- [ ] 장례 절차 안내 콘텐츠
- [ ] 반려동물 생애 기록 앨범 자동 생성
- [ ] 펫로스 위로 ai 상담 챗봇
